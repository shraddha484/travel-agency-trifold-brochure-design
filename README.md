# Travel Agency Trifold Brochure Design

This project features a **trifold brochure design** created for a travel agency called **WanderNest Travel**. The brochure promotes international travel packages and highlights special offers for travelers.

## Project Overview
The objective of this design was to create an attractive and informative brochure that encourages customers to explore travel packages and book their dream trips.

The brochure includes a **summer special offer with a 20% discount on international tours**, along with key benefits such as free airport pickup and a complimentary tour guide.

## Design Features
- Trifold brochure layout
- Travel-themed visuals and icons
- Clear information hierarchy
- Engaging promotional content
- Attractive color palette inspired by nature and travel destinations
- Call-to-action section for booking trips

## Sections Included
- Cover page with travel branding
- Promotional offer section
- Travel benefits and highlights
- Contact information and social media links
- Call-to-action for booking

## Tools Used
- Canva / Adobe Illustrator
- Typography and layout design
- Color theory
- Print design principles

## Purpose
This project demonstrates skills in **print design, brochure layout design, and marketing-focused visual communication**.
